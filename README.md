# Great Learning Courses Projects

Collection of scripts and projects developed during Great Learning courses.

This repository contains exercises, scripts, and small projects created as part of various courses attended at Great Learning.
It serves as a personal archive to track progress and consolidate knowledge across different topics.

## 🚀 How to Use

Clone the repository:

```bash
git clone git@github.com:eleoracca/GreatLearning.git
```

Navigate into a specific project or course folder to explore the code.

## 🛠 Requirements

Use the setup.sh script to prepare the environment:

```code
bash setup.sh setup
```

or to update the requirements:

```code
bash setup.sh update_requirements
```
